VsCode常用快捷键

mac 与 windows 键盘上功能键对应：

macwindows
CommandAlt
ControlCtrl
OptionWin

注释：

  • 单行注释：ctrl + /
  • 取消单行注释：ctrl+k,ctrl + u (按下 ctrl 不放，再按 k + u)
  • 多行注释：alt + shift + A
  • 多行注释：/**

移动行：option + up/down

显示/隐藏左侧目录栏 alt + b

向上/下复制当前行：shift + option +up/down

删除当前行：ctrl + CapsLock + k

控制台终端显示与隐藏：ctrl + ~

查找文件/安装 vs code 插件地址：alt + p

代码格式化：shift + alt +f

新建一个窗口 : alt + shift + n

行增加缩进: alt + [

行减少缩进: alt + ]

裁剪尾随空格(去掉一行的末尾那些没用的空格) : ctrl + shift + x

字体放大/缩小: ctrl + ( + 或 - )

切换到指定窗口 : ctrl + 1/2/3

切换到上一个编辑的窗口 : ctrl + Tab

关闭或开启拆分窗口 : ctrl + alt + left/right

关闭编辑器窗口 : alt + w

关闭所有窗口 : alt + q

切换到桌面 : F11

显示 git : ctrl + shift + g

全局查找/替换文件：alt + shift + h

显示相关插件的命令(如：git log)：alt + shift + p

选中文字：shift + left / right / up / down

删除行 ： alt + shift + k

快速回到顶部 ： alt + up

快速回到底部 : alt + down

选中代码 ： shift + 鼠标左键

多行同时添加内容（光标） ：command + option + up/down

鼠标位置向前删除一个字符：ctrl + h

打开最近打开的文件：ctrl + r

打开控制台：ctrl + ~（英文模式下）

打开新的控制台：ctrl + shift + ~
